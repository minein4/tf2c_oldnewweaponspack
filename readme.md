
# Old New Weapons Pack for TF2 Classified

Port of the [New-Old Weapons Pack for TF2c](https://gamebanana.com/mods/40389) by [SilentFox3](https://gamebanana.com/members/1354100)

All credits to the original authors to the project.

## Changes to weapons and other Changes

Some weapons have been changed due to crash issues or have been changed to their modern tf2 version.


## Installation

To install Old New Weapons Pack for TF2 Classified follow the instructions below.

1. Click on the code button
2. Click download zip
3. Open the tf2c_oldnewweaponspack-main.zip file and folder with the same name
4. Now on steam right click Team Fortress Classified, goto Manage > Browse Local Files
5. Then goto tf2classified > custom (make it if it doesnt exist) 
6. Then extract all the tf2c_oldnewweapons folders into the custom folder
